##### art

(the p flag is to offer the path of this picture and the s flag is mean the scale which default value is 0.3 )

~~the ASCII art is a little ugly~~

You can use the **art** command to convert the jpg image into ASCII art like this

~~~bash
go run main.go art  -p ./example/gopher.jpg -s 0.5
~~~
